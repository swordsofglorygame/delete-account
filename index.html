// 🔥 حط هنا رابط الـ GitHub Pages بتاعك اللي طلعلك في خطوة جيت هاب (سيب علامة الـ / في الآخر) const GITHUB_PAGE_URL = "https://yourusername.github.io/delete-account/"; // إيميل الإدارة اللي هيستقبل إشعار تأكيد الحذف const ADMIN_EMAIL = "swordsofglorygame@gmail.com"; // 1. دالة إرسال الإيميلات (بتبعث اللاعب لرابط جيت هاب النظيف) function sendDeleteEmails() { const sheet = SpreadsheetApp.getActiveSpreadsheet().getActiveSheet(); const data = sheet.getDataRange().getValues(); let count = 0; for (let i = 1; i < data.length; i++) { const player = data[i][0]; // العمود A: PlayerID const email = data[i][1]; // العمود B: Email let token = data[i][2]; // العمود C: Token let status = data[i][3]; // العمود D: Status const sent = data[i][5]; // العمود F: EmailSent if (!email) continue; if (sent === "Yes" || status === "Confirmed") continue; if (!token) { token = Utilities.getUuid(); sheet.getRange(i + 1, 3).setValue(token); } if (!status) { status = "Waiting"; sheet.getRange(i + 1, 4).setValue(status); } // بناء الرابط الموجه لجيت هاب مع التوكن const link = GITHUB_PAGE_URL + "?token=" + encodeURIComponent(token); const html = `

تأكيد حذف الحساب نهائياً

مرحباً بك، لتأكيد رغبتك في حذف حسابك نهائياً من لعبة Swords of Glory، يرجى الضغط على الزر أدناه:

`; try { GmailApp.sendEmail(email, "Swords of Glory - تأكيد طلب حذف حسابك", "الرجاء فتح الرابط: " + link, { htmlBody: html }); sheet.getRange(i + 1, 6).setValue("Yes"); count++; } catch(e) { sheet.getRange(i + 1, 6).setValue("Error: " + e.message); } } SpreadsheetApp.getUi().alert(`✅ تم إرسال ${count} رسالة تأكيد.`); } // 2. دالة الاستقبال الخلفية (API) - دي اللي جيت هاب هيكلمها في الخلفية من غير ما اللاعب يشوفها function doGet(e) { const token = e.parameter.token; let resultStatus = "invalid"; if (token) { const sheet = SpreadsheetApp.getActiveSpreadsheet().getActiveSheet(); const data = sheet.getDataRange().getValues(); for (let i = 1; i < data.length; i++) { if (String(data[i][2]) === String(token)) { if (data[i][3] === "Confirmed") { resultStatus = "already"; } else { sheet.getRange(i + 1, 4).setValue("Confirmed"); sheet.getRange(i + 1, 5).setValue(new Date()); // إشعار الإدارة GmailApp.sendEmail(ADMIN_EMAIL, "تم تأكيد طلب حذف حساب لاعب (عبر جيت هاب)", "", { htmlBody: `

` }); resultStatus = "ok"; } break; } } } // إرجاع النتيجة كـ JSON مع تفعيل الـ CORS عشان المتصفح بره جوجل يوافق يقراها const output = JSON.stringify({ status: resultStatus }); return ContentService.createTextOutput(output) .setMimeType(ContentService.MimeType.JSON); }
